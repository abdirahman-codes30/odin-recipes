# Coffee recipe website
## Project Description
This website contains a collection of coffee recipes. I built it to practice my HTML fundamentals while completing the Recipes project from The Odin Project. The focus of this project was to strengthen my understanding of HTML by creating multiple web pages, linking them together, and structuring content using semantic elements.

## Features
- Homepage with links to individual recipe pages.
- Dedicated page for each coffee recipe.
- Ingredients and step-by-step preparation instructions for every recipe.
- Responsive images that adapt to different screen sizes.
- Clean, semantic HTML structure for improved readability and accessibility.

## Pages Included
- index.html – Homepage

- latte-macchiato.html

- cappuccino.html

- vanilla-iced-latte.html

## Technologies used
- HTML5

## How to Run the Project
- Clone/download the project.
- Open index.html in a browser.
